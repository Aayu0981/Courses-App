This project is a Next.js-based Courses Web App designed to display short courses in a clean and user-friendly interface—similar to FutureLearn.
Users can explore different courses and open detailed pages with a single click.
Features

✔️ Interactive homepage with course listings
✔️ Clickable course cards using next/link
✔️ Detailed Short Courses Page
✔️ Modular components (ContainerDiv, Course Page component)
✔️ Fully responsive with Tailwind CSS
✔️ Image and content customization supported
